githubの使い方

githubの使い方の手順

  1. アカウントの登録
  2. Source Treeのインストール
  3. リポジトリの作成
  4. リポジトリを登録

githubの作成

・Repository name：リポジトリ名を入力
・Description：リポジトリの説明
・Public：無料プランの場合はPublicのみ利用のため、こちらを選択する

githubの更新の仕方

1.HTMLファイルを作成します

2.ステージングしコミットします

git add ファイル名

git commit -m "Initial commit"

3.GitHubへプッシュ

git push
