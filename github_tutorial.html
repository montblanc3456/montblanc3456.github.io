githubの使い方

githubの使い方の手順

  1. アカウントの登録
  2. Source Treeのインストール
  3. リポジトリの作成
  4. リポジトリを登録

アカウントの登録

まず、GitHubのトップページにアクセスします。
ユーザー名・メールアドレス・パスワードを入力してユーザー登録

Source Treeのインストール

Source Treeをインストールして、GitHubのアカウントを入力

リポジトリの作成

・Repository name：リポジトリ名を入力
・Description：リポジトリの説明
・Public：無料プランの場合はPublicのみ利用のため、こちらを選択する

リポジトリの登録

・Set up in Desktop：「GitHub Desktop」のダウンロードページに遷移
・HTTPS：作成したリポジトリにHTTPS経由で接続するアドレス表示
・SSH：作成したリポジトリにSSH経由で接続するアドレスを表示

コマンド意味
git addファイルを追加
git commitファイルをコミット
git push origin masterファイルを更新
